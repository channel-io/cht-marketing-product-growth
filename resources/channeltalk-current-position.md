## 2. 제품 포지셔닝

### 2.1 공식 포지셔닝

**"올인원 AI 비즈니스 메신저"**

```
┌─────────────────────────────────────────────────────────┐
│                      채널톡                              │
├─────────┬─────────┬─────────┬─────────┬─────────────────┤
│ 채팅 상담 │ 팀 메신저 │ 인터넷 전화│ AI 챗봇  │ CRM 마케팅      │
└─────────┴─────────┴─────────┴─────────┴─────────────────┘
```

### 2.2 핵심 강점

| 강점 | 설명 |
|------|------|
| **채팅 기반 원천 기술** | 라이브챗에서 시작, 양방향 대화가 DNA |
| **상담 + 마케팅 통합** | 별도 툴이 아닌 하나의 플랫폼 |

| **테크 중심 조직** | 직원의 50%가 개발자 |

## 4. 고객 프로필 분석

### 4.1 고객 규모 분포 (추정)

```
고객당 평균 MRR ~19,000원 = 주로 SMB/소상공인

15만 고객 × 19,000원/월 × 12개월 = ARR 360억원
```

| 세그먼트 | 추정 비중 | 특성 |
|----------|-----------|------|
| **소상공인/1인 기업** | 높음 | 저가 플랜, 기본 기능 |
| **SMB (스타트업, 소규모 이커머스)** | 중간 | 핵심 고객층 |
| **Mid-Market** | 낮음 | 성장 중 |
| **Enterprise** | 매우 낮음 | 일부 대기업 |

### 4.2 주요 고객 산업 (추정)

- 이커머스/온라인 리테일
- D2C 브랜드
- 스타트업/SaaS
- 오프라인 서비스 (예약, 상담 등)

## 6. 현재 상황 진단

### 6.1 강점 (Strengths)

- 상담 SaaS 국내 상위, 일본 1위
- 채팅 기반 양방향 소통 원천 기술
- AI 상담 선도 (ALF 시리즈)
- 한국/일본 현지화 (카카오, LINE)
- 테크 중심 조직, 빠른 개발력
- 15만 고객 기반

### 6.2 약점/고민 (Weaknesses)

- **타겟 불명확**: SMB? Mid-Market? Enterprise?
- **전략보다 직관 기반** 의사결정 구조
- **마케팅 기능 방향성** 불명확
- 고객당 MRR 낮음 (ARPU 확대 필요)
- 글로벌 마케팅 툴 대비 기능 깊이 부족

### 6.3 기회 (Opportunities)

- **"Conversational CRM/Marketing"** 차별화 포지션
- AI 에이전트 트렌드 (SalesAlf 등)
- 아시아 시장 (한국, 일본 강점 기반)
- 상담 고객 → 마케팅 업셀 경로

### 6.4 위협 (Threats)

- 글로벌 툴들의 AI 투자 가속
- Intercom 등 채팅 기반 경쟁사
- SMB 시장의 가격 민감도
- 범용 마케팅 기능 경쟁 시 열세

## 8. 전략 방향성 결정을 위한 판단 근거

> 리서치 → **판단/결정** 단계로 넘어갈 때 필요한 지표들. 팀 내 데이터 담당자와 협업 필요.

### 8.1 내부 데이터 기반 판단 지표

| 지표 | 확인할 질문 | 왜 중요한가 |
|------|-------------|-------------|
| **고객 세그먼트별 매출 비중** | SMB vs Mid-Market vs Enterprise 매출이 각각 얼마인가? | 어디서 돈을 버는지 = 어디에 집중할지 |
| **마케팅 기능 사용률** | 전체 고객 중 마케팅 기능 쓰는 비율은? | 현재 PMF가 있는지 |
| **기능별 Retention 영향** | 마케팅 기능 쓰는 고객의 이탈률 차이는? | 마케팅이 Lock-in에 기여하는지 |
| **ARPU by 기능** | 마케팅 쓰는 고객 vs 안 쓰는 고객 ARPU 차이 | 업셀 효과가 있는지 |
| **고객 요청/피드백** | 가장 많이 요청하는 마케팅 기능은? | 시장 수요 신호 |
| **경쟁 이탈 사유** | 마케팅 기능 때문에 떠나는 고객이 있는가? 어디로? | 경쟁 압박 수준 |

### 8.2 시장 기반 판단 지표

| 지표 | 확인할 질문 | 어떻게 파악 |
|------|-------------|-------------|
| **TAM/SAM/SOM** | 채널톡이 노릴 수 있는 시장 크기는? | 시장 리서치 |
| **경쟁 강도** | 각 세그먼트별 경쟁이 얼마나 치열한가? | 경쟁사 분석 |
| **Win/Loss Rate** | 특정 세그먼트에서 이기는가 지는가? | 세일즈 데이터 |
| **가격 민감도** | 타겟 고객이 얼마까지 낼 의향이 있는가? | 고객 인터뷰, 실험 |

### 8.3 역량 기반 판단 지표

| 지표 | 확인할 질문 |
|------|-------------|
| **개발 리소스** | 새 기능 개발에 투입 가능한 인력은? |
| **기술 자산** | 채팅/AI에서 쌓은 기술이 마케팅에 활용 가능한가? |
| **GTM 역량** | 새 세그먼트를 공략할 세일즈/마케팅 역량이 있는가? |
| **조직 정렬** | 새 방향에 대해 조직이 얼라인될 수 있는가? |

### 8.4 전략 옵션별 판단 기준

#### Option A: "채팅 강점 극대화" (Conversational Marketing)

| 이 방향이 맞다면... |
|---------------------|
| ✓ 채팅 사용률이 높고, 마케팅과 연결될 때 전환율이 높다 |
| ✓ 경쟁사가 이 영역을 잘 못하고 있다 |
| ✓ 기존 고객이 "채팅 + 마케팅 통합"을 원한다 |

#### Option B: "범용 마케팅 툴로 확장"

| 이 방향이 맞다면... |
|---------------------|
| ✓ 마케팅 기능 때문에 이탈하는 고객이 많다 |
| ✓ 이메일/푸시 등 기본 기능 요청이 많다 |
| ✓ Mid-Market 이상으로 가려면 테이블 스테이크가 부족하다 |

#### Option C: "SMB 유지, 깊이보다 넓이"

| 이 방향이 맞다면... |
|---------------------|
| ✓ SMB 매출 비중이 압도적이고, 거기서 성장 여지가 있다 |
| ✓ 상위 시장 경쟁이 너무 치열하다 |
| ✓ 현재 ARPU로도 시장 크기가 충분하다 |

#### Option D: "상위 시장 진출"

| 이 방향이 맞다면... |
|---------------------|
| ✓ Mid-Market/Enterprise에서 Win Rate가 올라가고 있다 |
| ✓ 대형 고객이 요청하는 기능 gap이 명확하다 |
| ✓ ARPU 확대가 성장의 핵심이다 |
