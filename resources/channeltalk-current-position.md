## 2. 제품 포지셔닝

### 2.1 공식 포지셔닝

**"올인원 AI 비즈니스 메신저"**

```
┌─────────────────────────────────────────────────────────┐
│                      채널톡                              │
├─────────┬─────────┬─────────┬─────────┬─────────────────┤
│ 채팅 상담 │ 팀 메신저 │ 인터넷 전화│ AI 챗봇  │ CRM 마케팅      │
```

### 2.2 핵심 강점

| 강점 | 설명 |
|------|------|

## 4. 고객 프로필 분석

### 4.1 고객 규모 분포 (추정)

```
고객당 평균 MRR ~19,000원 = 주로 SMB/소상공인

15만 고객 × 19,000원/월 × 12개월 = ARR 360억원
```

| 세그먼트 | 추정 비중 | 특성 |
|----------|-----------|------|
| **소상공인/1인 기업** | 높음 | 저가 플랜, 기본 기능 |
| **SMB (스타트업, 소규모 이커머스)** | 중간 | 핵심 고객층 |
| **Mid-Market** | 낮음 | 성장 중 |
| **Enterprise** | 매우 낮음 | 일부 대기업 |

### 4.2 주요 고객 산업 (추정)

- 이커머스/온라인 리테일
- D2C 브랜드
- 스타트업/SaaS
- 오프라인 서비스 (예약, 상담 등)

## 6. 현재 상황 진단

### 6.1 강점 (Strengths)

- 상담 SaaS 국내 상위, 일본 1위
- 채팅 기반 양방향 소통 원천 기술
- AI 상담 선도 (ALF 시리즈)
- 한국/일본 현지화 (카카오, LINE)
- 테크 중심 조직, 빠른 개발력
- 15만 고객 기반

### 6.2 약점/고민 (Weaknesses)

- **타겟 불명확**: SMB? Mid-Market? Enterprise?
- **전략보다 직관 기반** 의사결정 구조
- **마케팅 기능 방향성** 불명확
- 고객당 MRR 낮음 (ARPU 확대 필요)
- 글로벌 마케팅 툴 대비 기능 깊이 부족

### 6.3 기회 (Opportunities)

- **"Conversational CRM/Marketing"** 차별화 포지션
- AI 에이전트 트렌드 (SalesAlf 등)
- 아시아 시장 (한국, 일본 강점 기반)
- 상담 고객 → 마케팅 업셀 경로

### 6.4 위협 (Threats)

- 글로벌 툴들의 AI 투자 가속
- Intercom 등 채팅 기반 경쟁사
- SMB 시장의 가격 민감도
- 범용 마케팅 기능 경쟁 시 열세
