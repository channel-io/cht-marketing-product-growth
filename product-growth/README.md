# Product Growth

GTM/PO 관점의 제품 전략 리서치 및 실행 프로젝트를 모아둔 폴더.

## 구조

| 폴더 | 질문 | 내용 |
|------|------|------|
| `context/` | 지금 상황이 어떤가? | 시장, 내부, 고객 현황 |
| `references/` | 남들은 어떻게 했나? | 외부 사례 연구 |
| `direction/` | 앞으로 어디로 가나? | 미래 방향, 비전 |
| `synthesis/` | 그래서 결론은? | 종합 분석, 전략 결정 |
| `initiatives/` | 뭘 실행하나? | 구체적 실행 프로젝트 |
