## 파는 가치 (Value Proposition)

### 핵심 메시지

**"온라인 쇼핑에 오프라인 매장의 판매원을 제공한다"**

| 문제 | Rep AI 해결책 |
|------|--------------|
| 고객 획득 비용 60% 증가 (2017년 이후) | 기존 트래픽 전환율 향상 |
| 평균 전환율 1.3%에 불과 | **평균 22% 전환율 향상** |
| 장바구니 이탈 | Rescue Algorithm™으로 92% 정확도 이탈 예측 |
| 24/7 지원 비용 | 지원 문의 95%+ 자동화 |

### 3가지 핵심 기능

1. **Proactive 세일즈**
   - 이탈 직전 고객 감지 → 대화 시작 → 구매 유도
   - Rescue Algorithm™: 92% 정확도로 이탈 시점 예측

2. **개인화 추천**
   - 대화 맥락 기반 상품 추천
   - 채팅 내 장바구니 담기
   - 키워드가 아닌 대화 컨텍스트 이해

3. **지원 자동화**
   - 주문 조회 (WISMO)
   - 반품/교환 처리
   - 주문 취소/수정
   - 구독 관리

### 고객 성과 (Case Study)

| 고객 | 결과 |
|------|------|
| Body Align | AOV 41% 증가, 지원 업무 90% 감소, **16x ROI** |
| Snow | 상담사 응대 98% 감소, 추가 매출 **$200K+** |
| 평균 | 첫 달 **30x ROI** (공식 주장) |

## 데이터 해자 (Moat)

### 핵심 데이터 자산

| 데이터 유형 | 수집 방법 | 활용 |
|------------|----------|------|
| **행동 데이터** | 쇼핑객 브라우징 패턴 | 이탈 시점 예측 (Rescue Algorithm™) |
| **대화 데이터** | AI 컨시어지 대화 | 구매 의도 파악, 추천 정확도 향상 |
| **카탈로그 데이터** | Shopify 10초 동기화 | 실시간 재고/가격 반영 |
| **전환 데이터** | 구매 완료 추적 | 어떤 대화가 구매로 이어지는지 학습 |

### 플라이휠 구조

```
더 많은 Shopify 상점 사용
        ↓
더 많은 쇼핑객 행동 데이터 수집
        ↓
이탈 예측 정확도 향상 (현재 92%)
        ↓
전환율 향상 → 더 많은 상점 가입
        ↓
(반복)
```

### 기술적 차별화

| 요소 | 내용 |
|------|------|
| Behavioral AI | 실시간 + 과거 데이터로 이탈 지점 파악 |
| LLM 통합 | ChatGPT 기반 자연어 대화 |
| Shopify 깊은 통합 | 카탈로그, 주문, 고객 데이터 실시간 연동 |

### 진입 장벽

1. **플랫폼 전용**: Shopify 깊은 통합 → 경쟁자 진입 장벽
2. **데이터 축적**: 500+ 상점에서 수집한 행동 데이터
3. **Behavioral AI + LLM 조합**: 단순 챗봇과 차별화
