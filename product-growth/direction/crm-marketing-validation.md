## 2. 상담+마케팅 연결로 풀 수 있는 문제가 있나?

**상태**: ✅ 검증 완료

### 근거 자료

| 출처 | 내용 |
|------|------|
| [salesforce-meeting/summary.md](../context/customer/salesforce-meeting/summary.md) | CS-마케팅 연결 고리 섹션 |
| [addon-growth-patterns.md](../references/addon-growth-patterns.md) | 글로벌 성공/실패 패턴 |

### VOC에서 발견된 연결 니즈

| CS 관점 | 마케팅 활용 |
|---------|------------|
| 상담 데이터 축적 | "유입을 상담으로만 소모하지 않고 DB확보 → 전환/재방문 유도" |
| 단순 문의 자동화 | "인력을 줄이기보다, 더 중요한 업무(마케팅)에 집중" |
| 고객 정보 수집 | "고객정보 연동 및 세그먼트 분류를 통한 마케팅 자동화" |
| 장바구니 이탈 문의 | "장바구니 이탈 고객 리마인드 캠페인" |

### 글로벌 현실

| 발견 | 내용 |
|------|------|
| 성공 사례 없음 | 글로벌 CS툴에서 "상담 데이터 → 마케팅"으로 성공한 사례 없음 |
| 방향 전환 | Intercom, Zendesk, Freshworks 모두 마케팅 확장 포기/후순위 |
| 집중 영역 | 모두 AI 상담/서비스에 올인 (2024-2026) |

### 자동화 방향의 한계

고객이 원하는 건 "데이터라이즈처럼 딸깍 한 번에 되는 자동화"
→ 하지만 이 방향은:

- 전문 마케팅 툴과의 리소스 싸움
- 채널톡 Moat(상담 데이터) 활용 아님
- = 부가 기능 실패 패턴

### 결론

- VOC에서 니즈는 있으나, 글로벌은 이미 포기한 방향
- 자동화 방향으로 가면 전문 툴과 경쟁 → 못 이김
- **채널톡만 할 수 있는 것**을 찾아야 함

## 4. 현재 CRM 마케팅 기능의 성과/한계는?

**상태**: ✅ 검증 완료

### 근거 자료

| 출처 | 내용 |
|------|------|
| [Tableau - 피처별 Net MRR](https://tableau.dmz.channel.io/#/views/Sales/NetMRR?:iid=1) | 2024.1 ~ 2026.2 피처별 MRR 데이터 |

### Feature별 Net MRR (2026년 2월 기준)

| Feature | Net MRR | 비율 | 성장률 (2024.1 대비) |
|---------|---------|------|---------------------|
| mu (메시지 사용량) | 5.5억 | 22.1% | +154% |
| wf (워크플로우) | 5.3억 | 21.3% | +15% |
| opseat (운영 시트) | 4.9억 | 19.7% | +363% |
| **alf (AI)** | **3.4억** | **13.6%** | **신규 (2024.11~)** |
| **mkt (마케팅)** | **2.4억** | **9.8%** | **-12%** |
| sns | 1.1억 | 4.3% | 신규 |
| xms | 0.9억 | 3.8% | 신규 |
| 기타 | 1.3억 | 5.4% | - |
| **Total** | **25억** | 100% | - |

### 핵심 발견

1. **mkt는 전체의 ~10%** - 낮은 비중
2. **유일하게 역성장 (-12%)** - 다른 피처는 전부 성장 중
3. **ALF가 2개월 만에 mkt 추월** - 2024.11 런칭 → 2026.2 기준 3.4억 > 2.4억

### 비교: mkt vs alf

```
mkt (마케팅)
├── 비중: 9.8%
├── 성장: -12% (역성장)
└── 히스토리: 2024.1부터 존재

alf (AI)
├── 비중: 13.6%
├── 성장: 폭발 (2개월 만에 mkt 추월)
└── 히스토리: 2024.11 신규 런칭
```

### 시사점

- 회사 전략 방향(AI/BPO)이 **숫자로 증명**되고 있음
- mkt에 리소스 투입의 **기회비용이 명확**해짐
- 현재 mkt는 성장 동력 없이 **유지/감소 중**

### 결론

- "현재 성과가 낮다" + "역성장 중" 확인됨
- 동시에 AI(alf)는 폭발 성장 → 회사 방향과 일치
- mkt 단독 성장 전략은 기회비용 대비 정당화 어려움
- → 3번 검증에서 **상담 Moat + 마케팅** 연결점 찾아야 의미 있음

## 5. 고객사가 실제로 원하는 건 뭔가?

**상태**: ✅ 검증 완료

### 근거 자료

| 출처 | 내용 |
|------|------|
| [salesforce-meeting/summary.md](../context/customer/salesforce-meeting/summary.md) | 신규 고객 기대치 분석 |

### 신규 고객의 관심사 (1stOpp)

| 관심사 | 비율 |
|--------|------|
| 자동화 | 67% |
| CRM/마케팅 | 39% |
| 통합 | 34% |
| 도입 검토 | 34% |

### 기대 vs 현실 Gap

```
도입 시: "CRM 마케팅도 해보고 싶다" (39%)
    ↓
사용 중: "CS도 바쁜데 마케팅까지 손 못 댐" (62%)
    ↓
실제 사용: 시즌성 일괄발송만 (가장 단순한 형태)
```

### 핵심 발견

> **고객은 뭘 원하는지 모른다**

- 39%가 "원한다"고 했지만
- 마테크 이해도가 낮아서 **뭘 원하는지 구체화 못함**
- "쉬운 자동화"를 원하는 건데, 그건 전문 툴과 싸우는 방향

### 결론

- "원하는 것"을 물어봐도 답이 안 나옴
- 고객 대신 **뭘 해줄 수 있는지** 찾아야 함
- 그게 상담 Moat 기반이어야 차별화 가능
