# Conversational 타겟 선택 기준

> **목적**: Conversational Marketing/Sales 방향으로 갈 때, B2B vs B2C 중 어디를 우선 타겟으로 할지 판단하기 위한 기준

---

## 배경

Conversational Marketing/Sales는 B2B와 B2C 모두에서 적용 가능하나, 접근 방식이 다름:

| 구분 | B2B | B2C |
|------|-----|-----|
| **대표 사례** | Drift, Intercom | Rep AI |
| **가격 모델** | 고가 ($30k+/년) | 저가 ($39/월~) |
| **유통** | 엔터프라이즈 영업 | 앱스토어 |
| **핵심 기능** | 리드 자격검증 → 세일즈팀 핸드오프 | 이탈 방지 → 대화 내 구매 전환 |
| **연동** | Salesforce, HubSpot | Shopify |

→ 둘 다 가능하지만, **리소스 집중**을 위해 우선순위 필요

---

## 판단 기준

### 1. 현재 고객 구성

| 질문 | 확인 방법 |
|------|----------|
| B2B vs B2C 고객 비율은? | 고객 수 기준 |
| B2B vs B2C 매출 비율은? | MRR 기준 |
| 어느 쪽 성장률이 높은가? | YoY 비교 |

**판단**: 비율이 높고 성장하는 쪽이 기존 강점

### 2. Sales ALF 현재 방향

| 질문 | 확인 방법 |
|------|----------|
| Sales ALF PoC는 누구를 타겟으로 하는가? | 제품팀 확인 |
| B2B 패턴(리드→세일즈팀)인가, B2C 패턴(대화→구매)인가? | 기능 명세 확인 |

**판단**: 현재 PoC 방향과 일치하면 시너지

### 3. 제품 준비도

| 질문 | B2B 유리 | B2C 유리 |
|------|----------|----------|
| CRM 연동 | Salesforce, HubSpot 연동 있음 | - |
| 이커머스 연동 | - | Shopify, 카페24 연동 있음 |
| 결제 연동 | - | 대화 내 결제 가능 |

**판단**: 이미 갖춰진 인프라 쪽이 빠른 실행 가능

### 4. 팀 역량

| 질문 | 확인 방법 |
|------|----------|
| 팀에 B2B 세일즈 경험자가 있는가? | 팀 구성 확인 |
| 이커머스 마케팅 경험자가 있는가? | 팀 구성 확인 |

**판단**: 경험이 있는 쪽이 빠른 학습 곡선

### 5. 시장 접근 방식

| 질문 | B2B | B2C |
|------|-----|-----|
| 영업 방식 | 엔터프라이즈 세일즈 필요 | 앱스토어/셀프서브 가능 |
| 현재 채널톡 영업 방식과 맞는가? | 확인 필요 | 확인 필요 |

**판단**: 기존 GTM 모션과 맞는 쪽이 유리
