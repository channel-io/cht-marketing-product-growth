## 2. 데이터 기반 판단 프레임워크

### 2.1 시나리오 매트릭스

| 데이터 조합 | 시나리오 | 전략 방향 |
|-------------|----------|----------|
| 매출 비중 ↓ + 채택률 ↓ | 인지도/온보딩 문제 | UX 개선 우선 |
| 매출 비중 ↓ + 채택률 ↑ | 무료로 쓰고 있음 | 가격 구조 재검토 |
| 매출 비중 ↑ + 채택률 ↓ | 소수 헤비유저 | 확장 잠재력 있음 |
| 매출 비중 ↑ + 채택률 ↑ | 이미 성공 | 투자 확대 근거 |

### 2.2 핵심 질문

데이터를 통해 답해야 할 질문:

1. **현재 위치**: 마케팅 기능이 비즈니스에 얼마나 기여하고 있는가?
2. **성장 여력**: 채택률을 높일 여지가 있는가, 이미 포화인가?
3. **가치 인식**: 고객이 마케팅 기능에 돈을 낼 만큼 가치를 느끼는가?
4. **핵심 기능**: 어떤 마케팅 기능이 가장 많이 쓰이는가?
