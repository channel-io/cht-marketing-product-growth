## 6. 현재 상황 진단 (기능별)

### 6.1 상담 기능

| 구분 | 내용 |
|------|------|
| **현재 위치** | 국내 상위권, 일본 1위 (점유율 20%+) |
| **강점** | 채팅 기반 원천 기술, 한국/일본 현지화, 15만 고객 기반 |
| **방향** | AI 상담(ALF)으로 자동화 확대, 음성(VoiceAlf) 진출 |
| **성숙도** | 높음 - 핵심 캐시카우, PMF 검증됨 |

### 6.2 AI 기능 (ALF 시리즈)

| 구분 | 내용 |
|------|------|
| **현재 위치** | AI 상담 선도, 단순 문의 80% 처리 |
| **강점** | 빠른 개발력, 상담 데이터 축적, 테크 중심 조직 |
| **방향** | FrontAlf(상담) → VoiceAlf(음성) → SalesAlf(영업) 확장 |
| **성숙도** | 중간 - 성장 중, 투자 집중 영역 |

### 6.3 마케팅 기능 (CRM 마케팅) ⭐ 본 문서의 핵심

| 구분 | 내용 |
|------|------|
| **현재 위치** | 상담의 부가 기능, 전문 마케팅 툴 대비 후발 |
| **성숙도** | 낮음~중간 - 방향성 탐색 중 |

#### 강점
- 상담과 마케팅 통합 (발송 → 대화 → 전환 연결)
- 15만 고객 기반 = 업셀 잠재력
- 카카오톡/알림톡 등 한국 채널 지원
- 채팅 데이터 기반 타겟팅 가능성

#### 약점/고민
- **타겟 불명확**: 마케팅 기능의 ICP가 SMB인지, Mid-Market인지, 이커머스인지 정의 안 됨
- **방향성 부재**: 채팅 강점 살릴지, 범용 마케팅 툴로 갈지 결정 안 됨
- **기능 깊이 부족**: Braze, Klaviyo 등 전문 툴 대비 자동화/개인화 기능 열세
- **이메일 미성숙**: 핵심 채널인 이메일 기능 고도화 진행 중
- **전략 부재**: 직관 기반 킥오프 → 장기 프로젝트 구조로 인해 체계적 로드맵 부족

#### 기회
- **"Conversational Marketing"** - 대부분의 마케팅 툴이 못하는 영역
- 상담 고객 → 마케팅 업셀 (자연스러운 확장 경로)
- AI 에이전트 트렌드 (SalesAlf 등과 시너지)
- 한국/일본 SMB 시장 - 글로벌 툴들의 현지화 부족

#### 위협
- 글로벌 마케팅 툴(Braze, Klaviyo)의 아시아 진출 및 AI 투자
- Intercom 등 채팅 기반 경쟁사의 마케팅 기능 강화
- 범용 마케팅 기능 경쟁 시 리소스/기능 열세
- SMB 시장 가격 민감도 (ARPU 확대 한계)

### 6.4 기능 간 관계

```
[상담] ←──────→ [마케팅]
   │    고객 데이터    │
   │      공유       │
   ▼                ▼
[AI (ALF)] ──────→ 자동화/개인화 강화 가능성
```

**핵심 질문**: 상담/AI의 강점을 마케팅에 어떻게 연결할 것인가?

## 7. 핵심 질문

### 전략 방향성 결정을 위해 답해야 할 질문들

> **프레이밍**: A vs B 택일이 아니라, **어떤 가치를 우선해서 챙길 것인가 / 어떤 시장을 먼저 선점할 것인가**의 문제

| # | 영역 | 질문 |
|---|------|------|
| 1 | **타겟 시장** | 어떤 고객 세그먼트를 **먼저** 깊게 공략할 것인가? (SMB → Mid-Market? 특정 버티컬?) |
| 2 | **핵심 가치** | 고객에게 전달할 **첫 번째 가치**는 무엇인가? (채팅 연결? 자동화? 개인화? 쉬운 사용?) |
| 3 | **차별화 선점** | "Conversational Marketing" 포지션을 **먼저 선점**할 가치가 있는가? 시장이 존재하는가? |
| 4 | **기능 우선순위** | 제한된 리소스로 **가장 먼저** 투자할 영역은? (이메일 고도화 / AI 연동 / 새 채널 / 자동화 깊이) |
| 5 | **지역 집중** | 어느 시장에서 **먼저 이기고**, 그 다음 확장할 것인가? (한국 → 일본 → 글로벌?) |
| 6 | **경쟁 전략** | 전문 마케팅 툴과 **어느 영역에서** 싸울 것이고, **어느 영역은 피할** 것인가? |

## 8. 전략 방향성 결정을 위한 판단 근거

> 리서치 → **판단/결정** 단계로 넘어갈 때 필요한 지표들. 팀 내 데이터 담당자와 협업 필요.

### 8.0 채널톡 맥락에서의 관점

```
[일반 마케팅 툴]
고객이 "마케팅"을 목적으로 가입 → 마케팅 기능이 리텐션/이탈에 직접 영향

[채널톡]
고객이 "상담"을 목적으로 가입 → 마케팅은 업셀 대상
```

**따라서 핵심 질문은:**
- ~~마케팅 기능이 이탈을 막는가?~~ (X)
- **상담 고객을 어떻게 마케팅 사용자로 전환시킬 것인가?** (O)

```
[상담 목적 가입] → [상담 기능 사용] → [마케팅 기능 인지] → [마케팅 기능 사용] → [ARPU 확대]
                                    ↑
                              여기가 핵심 전환점
```

### 8.1 내부 데이터 기반 판단 지표

#### 상담 → 마케팅 전환 퍼널

| 지표 | 확인할 질문 | 왜 중요한가 |
|------|-------------|-------------|
| **마케팅 기능 사용률** | 전체 상담 고객 중 마케팅 기능 쓰는 비율은? | 현재 전환율 파악 |
| **전환 경로** | 어떤 계기로 마케팅 기능을 쓰기 시작했는가? | 전환 트리거 발견 |
| **미사용 사유** | 왜 마케팅 기능을 안 쓰는가? (모름 / 필요 없음 / 어려움 / 다른 툴 사용) | 전환 장벽 파악 |
| **사용 시작 시점** | 가입 후 얼마 만에 마케팅 기능을 쓰기 시작하는가? | 온보딩 타이밍 |

#### ARPU 확대 효과

| 지표 | 확인할 질문 | 왜 중요한가 |
|------|-------------|-------------|
| **ARPU 차이** | 마케팅 쓰는 고객 vs 안 쓰는 고객 ARPU 차이 | 업셀 효과 크기 |
| **플랜 업그레이드율** | 마케팅 사용자의 상위 플랜 전환율 | 수익 기여도 |
| **추가 결제** | 마케팅 관련 추가 비용 (발송량, 채널 등) | 수익 구조 |

#### 고객 세그먼트별 기회

| 지표 | 확인할 질문 | 왜 중요한가 |
|------|-------------|-------------|
| **세그먼트별 전환율** | 어떤 고객군이 마케팅을 더 많이 쓰는가? (규모, 산업, 사용 기간) | 타겟 우선순위 |
| **고객 요청/피드백** | 가장 많이 요청하는 마케팅 기능은? | 수요 신호 |
| **외부 툴 사용** | 마케팅 기능 안 쓰는 고객이 다른 툴을 쓰고 있는가? 어떤 툴? | 경쟁/대체재 파악 |

### 8.2 시장 기반 판단 지표

| 지표 | 확인할 질문 | 어떻게 파악 |
|------|-------------|-------------|
| **TAM/SAM/SOM** | "상담+마케팅 통합" 시장 크기는? | 시장 리서치 |
| **번들 수요** | 고객이 상담+마케팅 통합을 원하는가, 별도 툴을 선호하는가? | 고객 인터뷰 |
| **가격 민감도** | 마케팅 기능에 추가로 얼마까지 낼 의향이 있는가? | 고객 인터뷰, 실험 |
| **경쟁 구도** | 상담 고객을 마케팅으로 전환시키려 할 때 경쟁하는 툴은? | 경쟁사 분석 |

### 8.3 역량 기반 판단 지표

| 지표 | 확인할 질문 |
|------|-------------|

### 8.4 전략 옵션별 판단 기준

> **전제**: 고객은 상담 목적으로 가입. 마케팅은 업셀 대상. 핵심은 "어떻게 상담 고객을 마케팅 사용자로 전환시킬 것인가"

#### Option A: "채팅 강점 극대화" (Conversational Marketing)

**전략**: 상담↔마케팅 연결에서 차별화. 채팅 기반 마케팅이라는 새로운 카테고리 선점.

| 이 방향이 맞다면... |
|---------------------|
| ✓ 상담 데이터를 활용한 마케팅에서 높은 전환율/성과가 나온다 |
| ✓ 고객이 "상담과 마케팅이 연결되는 것"에 가치를 느낀다 |
| ✓ 경쟁사(Braze, Klaviyo 등)가 이 영역을 잘 못하고 있다 |

#### Option B: "범용 마케팅 기능 고도화"

**전략**: 상담 고객이 별도 마케팅 툴을 쓰지 않도록, 채널톡 내에서 마케팅 니즈를 해결.

| 이 방향이 맞다면... |
|---------------------|
| ✓ 상담 고객 중 상당수가 이미 별도 마케팅 툴(Mailchimp, Klaviyo 등)을 쓰고 있다 |
| ✓ "채널톡 마케팅으로는 부족해서" 다른 툴을 쓴다는 피드백이 있다 |
| ✓ 이메일/자동화 등 기본 기능 요청이 많고, 이걸 해결하면 전환될 고객이 있다 |

#### Option C: "현재 고객 기반에서 전환율 극대화"

**전략**: 새 기능 개발보다, 기존 상담 고객을 마케팅 사용자로 전환시키는 데 집중.

| 이 방향이 맞다면... |
|---------------------|
| ✓ 마케팅 기능 자체는 충분한데, 고객이 모르거나 안 써서 전환이 안 되는 상황 |
| ✓ 온보딩/교육/인앱 유도를 개선하면 전환율이 올라갈 여지가 크다 |
| ✓ 현재 기능으로도 쓰는 고객의 만족도/성과가 높다 |

#### Option D: "특정 버티컬/세그먼트 집중"

**전략**: 모든 상담 고객이 아닌, 마케팅 전환 가능성이 높은 특정 고객군에 집중.

| 이 방향이 맞다면... |
|---------------------|
| ✓ 특정 세그먼트(예: 이커머스, 특정 규모)에서 마케팅 전환율이 유독 높다 |
| ✓ 그 세그먼트의 니즈에 맞춘 기능 개발이 효율적이다 |
| ✓ "모든 고객"보다 "핵심 고객"에 집중하는 게 리소스 대비 효과적이다 |
| ✓ ARPU 확대가 성장의 핵심이다 |
