## 발견된 문제점

### 1. 진입 단계
- 사이드바에서 "마케팅" 메뉴 찾기는 어렵지 않았음
- 단, "캠페인"과 "마케팅 레시피"의 차이를 이해하는 데 시간 필요

### 2. 설정 복잡도
- 레시피 선택까지는 진행함
- **레시피가 있으면**: 메시지 수정 정도로 완료 가능 (테스트 2 성공)
- **레시피가 없으면**: 캠페인 편집 화면에서 막힘
  - 이벤트 조건, 대상 고객, 발송 시간, 메시지 내용 등 설정 항목이 너무 많음
  - 발송 트리거, 대상 세그먼트, 지연 시간 등을 어떻게 조합해야 할지 모름
- 초안 상태로 저장 후 더 이상 진행하지 못함

### 3. 이커머스 연동 부재 (신규 발견)
- 장바구니 담기(AddToCart) 같은 이커머스 이벤트가 없음
- 카페24/아임웹 연동이 되어 있지 않으면 핵심 시나리오 구현 불가
- 사용자가 직접 연동하는 방법을 모름
- **타겟 고객(카페24/아임웹 사업자)에게 치명적인 문제**

### 4. 용어 장벽
이해하지 못한 용어 목록:
- CRM
- 세그먼트
- 퍼널
- 트리거
- 퍼블리시
- 컨버전(전환)
- CTA(Call to Action)
- 리드수집 ("대충 고객 정보 모으는 건가?")
- 커맨드 버튼 ("링크 버튼이랑 뭐가 다르지?")
- 워크플로우
- A/B 테스트

### 5. 발송 불안감
- "레시피 사용하기", "퍼블리시" 버튼 누르면 바로 고객에게 발송될까 봐 걱정
- 초안 상태 표시로 일부 해소되었으나, 설정 실수로 이상한 메시지 나갈까 두려움 지속
- **테스트 발송 기능이 눈에 잘 안 띔** → 더 눈에 띄게 해달라는 피드백

### 6. 상담 기능 대비 체감 난이도
- 상담 기능: 채팅만으로 직관적 사용 가능
- 마케팅 기능: 설정 항목 많고 용어 낯설어 난도 높게 체감
- 고객 세그먼트 지정, 발송 조건 설정 등은 별도 공부 필요해 보임

## 최종 판단

### 테스트 1
> "당장은 안 쓸 것 같다. 마케팅 자동화로 전환율 높일 수 있다는 건 매력적이지만, 설정 과정이 복잡하고 용어 이해가 부족해 배울 시간이 필요하다. 여유가 생기면 다시 시도해볼 의향은 있다."

### 테스트 2
> "조건부로 계속 쓸 것 같다."

**쓸 것 같은 이유:**
- 레시피 덕분에 신규 방문자 쿠폰 팝업은 만들 수 있었음
- 미리보기가 바로 보여서 좋았음
- 한 번 세팅하면 자동으로 돌아가니까 편할 것 같음

**조건:**
- 장바구니 이탈자 같은 복잡한 건 누가 도와줘야 함
- 카페24 연동 방법을 알아야 함
- 처음에는 사용 가이드 영상이 필요함
- 30분 넘게 걸리면 바쁜 날은 포기할 것

### problem-definition.md 검증 결과

| 단계 | 상태 | 비고 |
|------|------|------|
| 뭘 해야 하는지 | 레시피가 해결 | 레시피 있으면 성공 |
| 어떻게 설정해야 하는지 | **레시피 없으면 막힘** | 직접 설정은 실패 |
| 용어 이해 | **이해 불가** | 11개 용어 모름 |
| 이커머스 연동 | **불가** | AddToCart 등 이벤트 없음 |

### 핵심 발견

1. **레시피가 핵심 해결책**
   - 레시피 있으면 → 완료 가능
   - 레시피 없으면 → 포기

2. **레시피 커버리지 부족**
   - 신규 전환: 레시피 있음 ✅
   - 장바구니 이탈: 레시피 없음 ❌

3. **카페24/아임웹 연동이 전제조건**
   - 연동 안 되면 이커머스 이벤트 사용 불가
   - 타겟 고객이 이 연동을 혼자 못함

4. **시간 제약**
   - 30분 이내에 완료 가능해야 함
   - 그 이상 걸리면 "바쁜 날은 포기"
