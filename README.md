# Core Marketing

채널톡 CRM 마케팅 제품 전략 리서치.

## 현재 상태

원래 온보딩 UX 개선 리서치로 시작했으나, **제품 방향성 결정이 우선**이라 판단하여 전략 리서치 진행 중.

## 구조

```
core-marketing/
├── product-growth/          # GTM/PO 관점 전략 리서치 + 실행
│   ├── context/             # 현재 상황 (시장, 내부, 고객)
│   ├── references/          # 외부 사례 연구
│   ├── direction/           # 미래 방향
│   ├── synthesis/           # 종합 결론
│   └── initiatives/         # 실행 프로젝트
│
└── methodology/             # 사고 방법론
```
