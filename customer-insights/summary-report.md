# 채널톡 마케팅 기능 고객 인식 종합 보고서

> **분석 기간**: 2025-01-24 ~ 2026-02-06 (약 1년)
>
> **총 분석 건수**: 3,676건 (Renewal 1,000 + Check-In 1,000 + PostSales/MI 676 + 1stOpp 1,000)
>
> **작성일**: 2026-01-30

---

## Executive Summary

채널톡 마케팅 기능에 대한 고객 인식을 4개 트랙(이탈, 사용현황, 온보딩, 신규기대)으로 분석한 결과, **마케팅 기능의 낮은 채택률은 기능 자체의 문제보다 구조적 장벽에 기인**함을 확인했습니다.

### 핵심 발견

| 발견 | 수치 | 의미 |
|------|------|------|
| CS 우선 현상 | 62% | 기존 고객 대부분이 "CS 먼저, 마케팅 나중에" |
| 자동화 기대 | 67% | 신규 고객의 핵심 관심사는 자동화 |
| 비용 민감도 | 25% | 이탈 논의의 1/4이 비용 관련 |
| 리소스 장벽 | 37% | 인력/시간 부족으로 마케팅 세팅 불가 |

## 트랙별 핵심 인사이트

### 트랙 A: 이탈 패턴 (Renewal)

| 이탈 사유 | 비율 | 핵심 인사이트 |
|----------|------|--------------|
| 비용 이슈 | 25% | "가장 비싼 SaaS" 인식, 비용 절감 압박 |
| 기능 불만 | 10% | 서비스 이슈, 원하는 기능 부재 |
| 경쟁 전환 | 8% | "저비용 대체재로 이관 가능" |
| 축소/해지 | 7% | 마케팅만 빼고 CS만 유지 시도 |
| 가치 미입증 | 4% | ROI 체감 부족, 전환율 불확실 |

**핵심 메시지**: 마케팅 기능의 가치를 증명하지 못하면 비용 절감 대상이 됨

---

### 트랙 B: 사용 현황 (Check-In)

| 미사용 사유 | 비율 | 핵심 인사이트 |
|------------|------|--------------|
| CS 우선 | 62% | "CS 먼저, 마케팅 나중에" 로드맵 |
| 리소스 부족 | 37% | 1인 실무자, 손 댈 여유 없음 |
| 세팅 복잡 | 36% | 방법 모름, 처음부터 다시 해야 |
| 우선순위 밀림 | 15% | "급하지 않다", "나중에" |

**핵심 메시지**: 마케팅 기능을 몰라서 안 쓰는 게 아니라, 쓸 여유가 없음

---

### 트랙 C: 온보딩 (PostSales/MI)

| 온보딩 패턴 | 비율 | 핵심 인사이트 |
|------------|------|--------------|
| AI/ALF 중심 | 58% | 온보딩 논의의 절반이 AI 관련 |
| 세팅 지원 요청 | 22% | 직접 세팅보다 도움받기 선호 |
| 활용 확장 | 16% | 외부 시스템 연동, 기능 확장 |
| 성과 측정 | 10% | 전환율, 해결률 기준 관심 |

**핵심 메시지**: 온보딩 단계에서 마케팅은 빠짐 → CS 안정화 후로 밀림

---

### 트랙 D: 신규 기대 (1stOpp)

| 관심사 | 비율 | 핵심 인사이트 |
|--------|------|--------------|
| 자동화 | 67% | CS 자동화가 최우선 관심 |
| CRM/마케팅 | 39% | 관심은 있으나 2순위 |
| 통합 | 34% | 전화+채팅+카카오 통합 니즈 |
| 도입 검토 | 34% | 명확한 도입 의향 |

**핵심 메시지**: 신규 고객은 CRM/마케팅에 기대를 갖고 도입하지만, 현실에서는 CS 우선으로 밀림

## 개선 방향 제안

### 1. 온보딩 단계에서 마케팅 함께 세팅

| 현재 | 개선 |
|------|------|
| CS 세팅 → (나중에) 마케팅 | CS + 마케팅 동시 세팅 |
| 고객이 직접 세팅 | 온보딩 전담팀이 기본 캠페인 세팅 |
| 마케팅은 옵션 | 마케팅 기본 템플릿 자동 활성화 |

### 2. 리소스 장벽 해소

| 현재 | 개선 |
|------|------|
| 세팅 복잡 | 원클릭 캠페인 템플릿 |
| 방법 모름 | 인앱 가이드/튜토리얼 |
| 1인 실무자 한계 | AI 기반 캠페인 자동 추천 |

### 3. 가치 증명 자동화

| 현재 | 개선 |
|------|------|
| ROI 체감 어려움 | 자동 성과 리포트 |
| 전환율 불확실 | 매출 연동 대시보드 |
| "마케팅 효과 없다" | 캠페인별 기여도 시각화 |

### 4. 비용 인식 개선

| 현재 | 개선 |
|------|------|
| "가장 비싼 SaaS" | 절감 효과 대비 비용 시각화 |
| 변동 비용 예측 불가 | 월별 예상 비용 대시보드 |
| 마케팅만 해지 시도 | 마케팅 ROI 증명으로 방어 |

## 결론

채널톡 마케팅 기능의 낮은 채택률은 **"기능의 문제"가 아닌 "구조의 문제"**입니다.

신규 고객의 39%가 CRM/마케팅에 관심을 갖고 도입하지만, 온보딩 단계에서 CS/ALF 중심으로 진행되고(58%), 사용 단계에서는 CS 우선으로 밀리며(62%), 갱신 시점에는 가치 미입증으로 해지/축소 대상이 됩니다.

이 구조를 깨기 위해서는 **온보딩 단계에서 마케팅을 함께 세팅**하고, **리소스 장벽을 낮추며**, **가치를 자동으로 증명**하는 시스템이 필요합니다.
